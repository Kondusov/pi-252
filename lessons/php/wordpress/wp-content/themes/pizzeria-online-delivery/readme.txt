== Description ==

Pizzeria Online Delivery is a WordPress theme designed for pizza shops, restaurants, cafés, fast-food outlets, and cloud kitchens that want to offer online food ordering through their website. The theme uses a clean layout and is built to work with the WordPress block editor, allowing users to manage content without relying on external page builders. The theme integrates with WooCommerce, enabling website owners to create an online menu, manage orders, accept payments, and handle food delivery or takeaway options. It includes dedicated sections for menus, offers, testimonials, and business information, making it suitable for both single-location and multi-location restaurants. Pizzeria Online Delivery is responsive and adapts well to mobile, tablet, and desktop devices. The theme is optimized for performance and follows SEO-friendly coding practices to support better visibility in search engines. It can be used for new restaurant websites or for upgrading existing food ordering sites that require a structured and easy-to-manage layout.

== Installation ==
        
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the pizzeria-online-delivery theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away
